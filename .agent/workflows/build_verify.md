---
description: Arranca la web y verifica que los flujos principales funcionan correctamente.
---

1. Arrancar servidor de desarrollo
   Ejecutar `npm run dev` y comprobar que carga sin errores en consola.

2. Validar flujos principales
   Recorrer la web en el navegador:
   - Navegar por todas las páginas.
   - Probar todos los links de navegación.
   - Probar los formularios (envío, validación, estados de error/éxito).
   - Verificar que las animaciones se ejecutan correctamente.
   - Comprobar modo móvil (resizear a 375px).

3. Documentar hallazgos
   Actualizar `docs/walkthrough.md` con:
   - Screenshots de las páginas principales.
   - Lista de issues encontrados (si los hay).
   - Confirmación de que los flujos críticos funcionan.
