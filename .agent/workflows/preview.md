---
description: Arranca la web y captura screenshots para previsualizar con el cliente.
---

1. Arrancar el servidor de desarrollo
   Ejecutar `npm run dev` (o el comando equivalente del proyecto).
   Esperar a que cargue correctamente sin errores.

2. Capturar screenshots de Desktop
   Abrir el navegador con el viewport en 1440px de ancho.
   Navegar a cada página del sitio y capturar screenshot completo (full page).

3. Capturar screenshots de Móvil
   Redimensionar el navegador a 375px de ancho.
   Navegar a cada página y capturar screenshot completo.

4. Recopilar evidencia
   Guardar todos los screenshots en una carpeta organizada.
   Crear o actualizar `docs/walkthrough.md` con las capturas embebidas.

5. Resumen para el cliente
   Informar al usuario con:
   - Screenshots de cada página (desktop y móvil).
   - Lista de secciones implementadas.
   - Lista de pendientes (si los hay).
   - Preguntas para el usuario si necesita feedback del cliente.
