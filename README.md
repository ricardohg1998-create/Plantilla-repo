> High-performance, anti-generic web template for professionals.

## Tech Stack
- **Framework**: Next.js 15+ (App Router) / Vite (React)
- **Styling**: Tailwind CSS v4 + Framer Motion
- **Language**: TypeScript (Strict)
- **Architecture**: Monorepo (npm workspaces)

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Start Development Server**:
    ```bash
    npm run dev
    ```

3.  **Check Content Safety**:
    ```bash
    npm run check:content
    ```

## Project Structure
- `starters/next`: The main SSR web application.
- `starters/vite`: The SPA alternative.
- `shared`: Reusable UI components and logic (The "Core").
- `docs`: Architecture decisions and guides.

## Key Features
- **Anti-generic UI**: Custom components in `shared/ui`.
- **Content Safety**: Automated checks for "Lorem Ipsum".
- **SEO Ready**: Automated sitemap, robots, and JSON-LD.
- **A11y First**: Accessible patterns and reduced motion support.
